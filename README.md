# 🐍 Snake Game

A modern, polished take on the classic **Snake** game — built with pure **HTML, CSS, and JavaScript** in a single file. No frameworks, no build step. Just open and play.

The game features a glowing neon board, smooth animations, particle bursts when you eat food, increasing speed as your score grows, and a saved best score.

## 🎮 Live Demo

▶️ **Play it here:** [https://faizankhan17623.github.io/Snake-Game/](https://faizankhan17623.github.io/Snake-Game/)

---

## ✨ Features

- 🎮 **Smooth gameplay** rendered on HTML5 Canvas
- ⚡ **Dynamic speed** — the snake speeds up as your score climbs
- 🏆 **Best score saved** in your browser (`localStorage`)
- 💥 **Particle effects** when the snake eats food
- ⏸️ **Pause / Resume** support
- 📱 **Mobile friendly** — on-screen direction pad for touch devices
- ⌨️ **Keyboard controls** — Arrow keys or WASD
- 🎨 **Modern UI** with glassmorphism panels and a neon glow theme

---

## 🕹️ Controls

| Action | Keys |
|--------|------|
| Move   | `↑` `↓` `←` `→` or `W` `A` `S` `D` |
| Start / Pause / Resume | `Space` |
| Mobile | On-screen direction pad |

> Tip: The snake speeds up as your score climbs — keep your run clean!

## 🚀 How to Run

No installation needed — it's a single HTML file.

**Easiest way:** just open the [Live Demo](https://faizankhan17623.github.io/Snake-Game/) and play in your browser.

**Or run it locally:**

1. Clone the repo:
   ```bash
   git clone https://github.com/Faizankhan17623/Snake-Game.git
   ```
2. Open the folder and double-click **`index.html`** (or open it in any browser).

That's it — start playing! 🎉
